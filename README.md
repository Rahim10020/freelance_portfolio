## 🛠️ Technologies

- **Framework** : Next.js 15.5.5 (App Router)
- **UI Library** : React 19.1.0
- **Language** : TypeScript 5.x
- **Styling** : Tailwind CSS 4.x
- **Icons** : Custom SVGs
- **Deployment** : Vercel
